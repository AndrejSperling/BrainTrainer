### Installation

```sh
$ npm install
```

### Build

```sh
$ npm run build
```

### Webansicht

Starte Webansicht:

```sh
$ npm run start-web
```

### Electron

Starte Electron:

```sh
$ npm run start-electron
```

Erzeuge ausführbare Dateien

```sh
$ npm run pack-electron
```

### Veränderungen beobachten

Änderungen werden sofort übertragen:

```sh
npm run gulp watch
```
